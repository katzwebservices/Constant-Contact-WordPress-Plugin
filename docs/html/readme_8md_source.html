51 == Installation ==
52 
53 To install the plugin follow the steps below:
54 
55 1. Upload `constant-contact-api` to the `/wp-content/plugins/` directory.
56 2. Activate the plugin through the 'Plugins' menu in WordPress.
57 3. Activate the Constant Contact API: Form Designer plugin (optional)
58 4. Click the new main menu item called "Constant Contact".
59 5. You'll need to enter your username and password on the settings page then save the page to see your contact lists.
60 6. Now Configure the "Register Page Settings" to get the checkbox or list selection displayed on the user register page.
61 
62 ### Using the Form Designer
63 1. Install this plugin.
64 2. Activate the Constant Contact API: Form Designer plugin
65 3. Configure the settings on the form designer by updating the settings in the boxes on the left.
66 4. Next to "Form Name" where it says "Enter form name here," enter your form name.
67 5. Once you have configured and named your form, click Save Form.
68 6. In the Appearance menu of the administration, click the Widgets link.
69 7. Drag the widget named "Constant Contact Form Designer" into the sidebar.
70 8. Configure the settings shown, then click the "Save" button at the bottom of the widget form.
71 9. You will see the signup widget you created on your website!
72 10. To edit the form, return the the Form Designer page (from Step 3) and click on the form tab with the name of the form you would like to edit. Edit the form, then click Update Form. The form will show as updated on your website.

566 = Is there shortcode support? =
567 
568 ### Form Shortcode ###
569 
570 There is shortcode support for the Form Designer forms: `[constantcontactapi]` with the following options:
571 
572 <pre>
573 'formid' => 0, // REQUIRED
574 'before' => null,
575 'after' => null,
576 'redirect_url' => false,
577 'lists' => array(),
578 'title' => '',
579 'exclude_lists' => array(),
580 'description' => '',
581 'show_list_selection' => false,
582 'list_selection_title' => 'Add me to these lists:',
583 'list_selection_format' => 'checkbox'
584 </pre>
585 
586 So to add a form, you would add the following in your content: `[constantcontactapi formid="3"]`
587 
588 ### Event Shortcode ###
589 
590 To show event details, you can use the `[eventspot]` shortcode with the following options:
591 
592 <pre>
593 'id' => null, // Show a specific event; enter Event ID (found on the Events page) to use
594 'limit' => 3, // Number of events to show by default
595 'showdescription' => true, // Show event Description
596 'datetime' => true, // Show event Date & Time
597 'location' => false, // Show event Location
598 'map' => false, // Show map link for Location (if Location is shown)
599 'calendar' => false, // Show "Add to Calendar" link
600 'directtoregistration' => false, // Link directly to registration page, rather than event homepage
601 'newwindow' => false, // Open event links in a new window
602 'style' => true // Use plugin styles. Disable if you want to use your own styles (CSS)
603 </pre>
604 
605 <strong>Sample Event Shortcodes</strong>
606 
607 * To show event details for 5 events using the default settings, you would use `[eventspot limit=3]`
608 * To show event details for a single event with the id of `abc123` and also show the location details and map link, you would use: `[eventspot id="abc123" location=true map=true]`
609 * To use your own CSS file, you would use `[eventspot style=false]`

612 = How do I use the new `apply_filters()` functionality? (Added 1.1) =
613 If you want to change some code in the widget, you can use the WordPress `add_filter()` function to achieve this.
614 
615 You can add code to your theme's `functions.php` file that will modify the widget output. Here's an example:
616 <pre>
617 function my_example_function($widget) {
618  // The $widget variable is the output of the widget
619  // This will replace 'this word' with 'that word' in the widget output.
620  $widget = str_replace('this word', 'that word', $widget);
621  // Make sure to return the $widget variable, or it won't work!
622  return $widget;
623 }
624 add_filter('constant_contact_form', 'my_example_function');
625 </pre>
626 
627 You can modify the widget output by hooking into any of the filters below in a similar manner.
628 
629 To modify the Events widget output, start with the following code, again in your theme's `functions.php` file:
630 
631 <pre>
632 add_filter('cc_event_output_single', 'cc_event_output_single', 1, 2);
633 
634 function cc_event_output_single($output, $pieces = array('start'=> '','title'=>'','description'=>'','date'=>'','calendar'=>'','location' => '', 'end'=>'')) {
635  // The pieces of each event are stored in the $pieces array
636  // So you can modify them and cut and paste in what order you
637  // want the pieces to display
638  return $pieces['start'].'<dt>Description</dt>'.$pieces['description'].$pieces['date'].$pieces['end'];
639 }
640 </pre>
641 
642 <strong>Some example filters:</strong>
643 
644 * Entire form output: `constant_contact_form`
645 * Successful submission message: `constant_contact_form_success`
646 * Form description text: `constant_contact_form_description` (after it has been modified by `wpautop()`)
647 * Error message: `constant_contact_form_errors`
648 * Submit button: `constant_contact_form_submit` (includes entire `input` string)

650 = My email campaign click data isn't being tracked in Constant Analytics =
651 Constant Contact does not have built-in Google Analytics "tagging" that would track the click data. When you create links in your Constant Contact campaigns, <strong><a href="http://www.google.com/support/analytics/bin/answer.py?answer=55578" rel="nofollow">use the Google URL Builder</a></strong> to add tags to your links. <strong>Make sure to set the Campaign Medium to `email`!</strong>
652 
653 When you do that, email click stats will be segmented for you in the Site Traffic box.

655 = How do I use the Form Designer? =
656 
657 ### Using the new Form Designer
658 1. Install this plugin.
659 2. Activate the Constant Contact API: Form Designer plugin
660 3. Configure the settings on the form designer by updating the settings in the boxes on the left.
661 4. Next to "Form Name" where it says "Enter form name here," enter your form name.
662 5. Once you have configured and named your form, click Save Form.
663 6. In the Appearance menu of the administration, click the Widgets link.
664 7. Drag the widget named "Constant Contact Form Designer" into the sidebar.
665 8. Configure the settings shown, then click the "Save" button at the bottom of the widget form.
666 9. You will see the signup widget you created on your website!
667 10. To edit the form, return the the Form Designer page (from Step 3) and click on the form tab with the name of the form you would like to edit. Edit the form, then click Update Form. The form will show as updated on your website.

669 = Form Designer isn't showing up or working =
670 Form Designer needs to be activated separately from the main plugin (see "How do I use the Form Designer?" above). Once you activate it, if it's still not working, it's likely a server issue.
671 
672 The problem is that your web server may think that Form Designer is an unwelcome script. In order to fix this, you should <strong>contact your web host and request that they "whitelist your domain for ModSecurity."</strong>.
673 
674 <a href="http://www.hostgator.com" rel="nofollow">HostGator</a> reps said whitelisting your own domain is <strong>not an issue that affects website security</strong>.
