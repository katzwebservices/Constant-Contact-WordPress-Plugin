39 == Installation ==
40 
41 To install the plugin follow the steps below:
42 
43 1. Upload `constant-contact-api` to the `/wp-content/plugins/` directory.
44 2. Activate the plugin through the 'Plugins' menu in WordPress.
45 3. Activate the Constant Contact API: Form Designer plugin (optional)
46 4. Click the new main menu item called "Constant Contact".
47 5. You'll need to enter your username and password on the settings page then save the page to see your contact lists.
48 6. Now Configure the "Register Page Settings" to get the checkbox or list selection displayed on the user register page.
49 
50 ### Using the Form Designer
51 1. Install this plugin.
52 2. Activate the Constant Contact API: Form Designer plugin
53 3. Configure the settings on the form designer by updating the settings in the boxes on the left.
54 4. Next to "Form Name" where it says "Enter form name here," enter your form name.
55 5. Once you have configured and named your form, click Save Form.
56 6. In the Appearance menu of the administration, click the Widgets link.
57 7. Drag the widget named "Constant Contact Form Designer" into the sidebar.
58 8. Configure the settings shown, then click the "Save" button at the bottom of the widget form.
59 9. You will see the signup widget you created on your website!
60 10. To edit the form, return the the Form Designer page (from Step 3) and click on the form tab with the name of the form you would like to edit. Edit the form, then click Update Form. The form will show as updated on your website.

554 = Is there shortcode support? =
555 
556 ### Form Shortcode ###
557 
558 There is shortcode support for the Form Designer forms: `[constantcontactapi]` with the following options:
559 
560 <pre>
561 'formid' => 0, // REQUIRED
562 'before' => null,
563 'after' => null,
564 'redirect_url' => false,
565 'lists' => array(),
566 'title' => '',
567 'exclude_lists' => array(),
568 'description' => '',
569 'show_list_selection' => false,
570 'list_selection_title' => 'Add me to these lists:',
571 'list_selection_format' => 'checkbox'
572 </pre>
573 
574 So to add a form, you would add the following in your content: `[constantcontactapi formid="3"]`
575 
576 ### Event Shortcode ###
577 
578 To show event details, you can use the `[eventspot]` shortcode with the following options:
579 
580 <pre>
581 'id' => null, // Show a specific event; enter Event ID (found on the Events page) to use
582 'limit' => 3, // Number of events to show by default
583 'showdescription' => true, // Show event Description
584 'datetime' => true, // Show event Date & Time
585 'location' => false, // Show event Location
586 'map' => false, // Show map link for Location (if Location is shown)
587 'calendar' => false, // Show "Add to Calendar" link
588 'directtoregistration' => false, // Link directly to registration page, rather than event homepage
589 'newwindow' => false, // Open event links in a new window
590 'style' => true // Use plugin styles. Disable if you want to use your own styles (CSS)
591 </pre>
592 
593 <strong>Sample Event Shortcodes</strong>
594 
595 * To show event details for 5 events using the default settings, you would use `[eventspot limit=3]`
596 * To show event details for a single event with the id of `abc123` and also show the location details and map link, you would use: `[eventspot id="abc123" location=true map=true]`
597 * To use your own CSS file, you would use `[eventspot style=false]`

600 = How do I use the new `apply_filters()` functionality? (Added 1.1) =
601 If you want to change some code in the widget, you can use the WordPress `add_filter()` function to achieve this.
602 
603 You can add code to your theme's `functions.php` file that will modify the widget output. Here's an example:
604 <pre>
605 function my_example_function($widget) {
606  // The $widget variable is the output of the widget
607  // This will replace 'this word' with 'that word' in the widget output.
608  $widget = str_replace('this word', 'that word', $widget);
609  // Make sure to return the $widget variable, or it won't work!
610  return $widget;
611 }
612 add_filter('constant_contact_form', 'my_example_function');
613 </pre>
614 
615 You can modify the widget output by hooking into any of the filters below in a similar manner.
616 
617 To modify the Events widget output, start with the following code, again in your theme's `functions.php` file:
618 
619 <pre>
620 add_filter('cc_event_output_single', 'cc_event_output_single', 1, 2);
621 
622 function cc_event_output_single($output, $pieces = array('start'=> '','title'=>'','description'=>'','date'=>'','calendar'=>'','location' => '', 'end'=>'')) {
623  // The pieces of each event are stored in the $pieces array
624  // So you can modify them and cut and paste in what order you
625  // want the pieces to display
626  return $pieces['start'].'<dt>Description</dt>'.$pieces['description'].$pieces['date'].$pieces['end'];
627 }
628 </pre>
629 
630 <strong>Some example filters:</strong>
631 
632 * Entire form output: `constant_contact_form`
633 * Successful submission message: `constant_contact_form_success`
634 * Form description text: `constant_contact_form_description` (after it has been modified by `wpautop()`)
635 * Error message: `constant_contact_form_errors`
636 * Submit button: `constant_contact_form_submit` (includes entire `input` string)

When you create links in your Constant Contact campaigns, <strong><a href="http://www.google.com/support/analytics/bin/answer.py?answer=55578" rel="nofollow">use the Google URL Builder</a></strong> to add tags to your links. <strong>Make sure to set the Campaign Medium to `email`!</strong>

643 = How do I use the Form Designer? =
644 
645 ### Using the new Form Designer
646 1. Install this plugin.
647 2. Activate the Constant Contact API: Form Designer plugin
648 3. Configure the settings on the form designer by updating the settings in the boxes on the left.
649 4. Next to "Form Name" where it says "Enter form name here," enter your form name.
650 5. Once you have configured and named your form, click Save Form.
651 6. In the Appearance menu of the administration, click the Widgets link.
652 7. Drag the widget named "Constant Contact Form Designer" into the sidebar.
653 8. Configure the settings shown, then click the "Save" button at the bottom of the widget form.
654 9. You will see the signup widget you created on your website!
655 10. To edit the form, return the the Form Designer page (from Step 3) and click on the form tab with the name of the form you would like to edit. Edit the form, then click Update Form. The form will show as updated on your website.

657 = Form Designer isn't showing up or working =
658 Form Designer needs to be activated separately from the main plugin (see "How do I use the Form Designer?" above). Once you activate it, if it's still not working, it's likely a server issue.
659 
660 The problem is that your web server may think that Form Designer is an unwelcome script. In order to fix this, you should <strong>contact your web host and request that they "whitelist your domain for ModSecurity."</strong>.
